## Contributing

- Commits should be descriptive of the changes made, following the conventional commit style. see https://www.conventionalcommits.org/.

  - Example: `feat: add new feature`, `fix: correct typo in documentation`
- Make sure to update the index file when adding new resources or materials.
- Add a good description to each resource or material added for clarity.
  - No harm in having assistance from AI, But at least a minimum of human input is required.
  - AI can summarize, I know, but a human touch is appreciated.
